## docker build
- 作用：
	- 根据dockerfile创建镜像
- 命令格式：
	- docker build [OPTIONS] PATH | URL | -
- 命令参数：


```
PATH		Dockerfile所在路径(文件夹路径)，文件名必须是Dockerfile
URL		Dockerfile所在URL地址

OPTIONS:
	-t, --tag list		为镜像设置名称和tag
	-f, --file string		指定Dockerfile的路径(这是可以使用其他名称命名Dockerfile)

```
